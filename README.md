# Giphy

# Projct Includes
1. Displaying all trending Gifs on Homescreen
2. Search gif using users input and show gif related to that keyword
3. Infinite scroll. rendering new gifs if user scrolling down to end
4. Pause and play gif
5. Theme change

#tested on
1. IOS using Expo
